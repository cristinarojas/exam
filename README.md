# exam

1. Clone the repository:

`git clone https://github.com/cristinarojas/exam.git`

2. Go to the project folder and then execute:

`bower install`

3. Open the index.html in your browser

# Supported Browsers

- Chrome
- Firefox
- Safari

# Supported Resolutions

- 1440x900
- 1366x768
- 1280x800
- 1280x768
- 1024x768

# Supported Devices

- iPad 2
- iPad 3
- iPad Air
- iPad Retina
